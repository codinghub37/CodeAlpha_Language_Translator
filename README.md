# 🌍 CodeAlpha Language Translator

## 📌 CodeAlpha AI Internship — Task 01

An AI-powered Language Translation Tool built using Python and Streamlit. This application allows users to translate text into 49+ languages instantly, copy translated text, and listen to translations using Text-to-Speech technology.

---

## ✨ Features

* 🌐 Translate text into 49+ languages
* ⚡ Real-time language translation
* 🔊 Text-to-Speech (TTS) support
* 📋 One-click copy functionality
* 🎨 Modern and responsive Streamlit interface
* 🚀 Fast and user-friendly experience

---

## 🛠️ Tech Stack

| Component            | Technology                         |
| -------------------- | ---------------------------------- |
| Programming Language | Python                             |
| Web Framework        | Streamlit                          |
| Translation Engine   | Deep Translator (Google Translate) |
| Text-to-Speech       | gTTS                               |

## ⚙️ How It Works

1. Select the source language.
2. Select the target language.
3. Enter or paste text into the input box.
4. Click the **Translate** button.
5. View the translated output instantly.
6. Copy the translated text or listen using the voice feature.
